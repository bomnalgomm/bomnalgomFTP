hidden(히든처리)

display:none

  • display:none은 모니터에도 안보이고 보조공학기기에서도 읽히지 않음
  • display:none, width:0, height:0; 속성 가운데 하나라도 포함되면 보조기기가 읽지 않고 건너뛸수 있으니 피해야함

visibility:hidden

  • 모니터에서만 보이지 않고 보조공학기기에서는 읽혀짐
					
페이지 제목


					/* 히든처리 */
					h1.hidden {visibility:hidden; width:1px; height:1px; overflow:hidden; position:absolute; left:-9999px;}
