작업시 유의사항

기능버튼

기능버튼 사용시 .fn-wrap과 셋트로 사용. 기능버튼은 button으로 마크업

게시판 하단 버튼

.btn-wrap은 기본 좌측 정렬이고 클래스 c(중앙정렬), r(우측정렬)을 이용하여 가로정렬 제어

페이지 이동관련 버튼은 a태그로, 값 전송부분은 input 태그로 마크업

게시판

게시판리스트 작업시 caption에 .blind 클래스로 히든처리

게시판 항목별 가로폭 조절은 colgroup내의 col style="width:10%" width의 % 값으로 조절

모달팝업

모달팝업구현시 포커스가 모달팝업으로 in되도록 구현 * 샘플 : http://webhipster.com/testing/accessibility/modal-dialog-latest/

정렬

.l(좌측), .c(중앙), .r(우측)

gap 사용

.gap : 30px, .sgap : 20px, .ssgap : 10px
