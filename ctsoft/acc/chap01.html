적절한 대체 텍스트 제공

이미지에 대한 대체 텍스트

img 태그를 사용하는 경우 alt 속성을 이용하여 대체 텍스트를 제공한다.

버튼 이미지에 대한 대체 텍스트

이미지를 이용한 입력 버튼은 input 태그를 사용하여 구현하고 버튼에 사용된 이미지의 대체 텍스트는 alt 속성을 이용하여 제공한다.

자바 애플릿 등에 대한 대체 텍스트 (applet, embed, object)

동영상이나 플래시 object를 제공할 경우 title 속성을 이용하여 대체 텍스트를 제공한다.
applet, embed 는 html 버전에 따라 표준정의 되지 않은 요소로 object 태그를 권장함 (크로스 브라우징 포함)

이미지 맵에 대한 대체 텍스트

area 태그별로 alt속성을 이용하여 해당 영역의 용도나 목적을 대체 텍스트로 제공한다.

중요한 정보에 대한 설명

대체 텍스트에는 색, 위치, 모양 등을 묘사하는 단어를 사용하지 않고 그 차이를 직접 문장으로 표시해야 한다.

QR 코드 대체 텍스트

QR코드의 대체 텍스트는 접속할 수 있는 웹사이트 주소를 대체 텍스트로 제공해야 한다. (웹사이트 주소를 a태그로 링크 걸어줘도 좋음)

자막제공

멀티미디어 콘텐츠를 동등하게 인식할 수 있도록 자막, 원고 또는 수화를 제공해야 한다.

적용

  • 동영상 제공되는 페이지에는 동영상 원고도 함께 텍스트로 제공하고 textarea 태그가 아닌 div 태그로 마크업 했을 경우 키보드 포커스가 원고 div 태그안에 이동될 수 있도록 tabindex="0"을 함께 마크업 [바로가기]

색에 무관한 콘텐츠 인식

콘텐츠에서 제공하는 모든 정보는 색을 배제하여도 특정한 색을 구별할 수 없는 사용자나 흑백 디스플레이 사용자가 해당 콘텐츠를 인식할 수 있도록 제공해야 한다.

명확한 지시사항 제공

특정 요소를 가리키거나 지시사항을 전달하는 콘텐츠에 적용하는 것으로 시각, 청각 등과 같은 특정 감각에만 의존하여 제공하면 안된다.

적용

텍스트 콘텐츠의 명도 대비

저시력자, 색각이상자, 고령자 등도 콘텐츠 인식할 수 있도록 핵심텍스트는 충분한 대비를 제공해야 한다.

  • 명도대비 4.5:1일 경우 (18pt 미만, 또는 굵은 14pt 미만)
  • 명도대비 3:1일 경우 (18pt 이상, 또는 굵은 14pt 이상)
